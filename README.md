# Task Manager Web App

## About the Project

This is a simple **Task Manager web application** built using Python and Flask.
The goal of this project is to help users organize their daily tasks in an easy way.

Users can add tasks, update them, mark them as completed, and delete tasks when they are finished.

This project was created to practice **backend web development using Flask** and to understand how web applications work.

---

## Features

* User login system
* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Simple task dashboard

---

## Technologies Used

* Python
* Flask
* HTML
* CSS
* SQLite (for storing tasks)

## How to Run the Project

1. Clone this repository

```
git clone https://github.com/yourusername/task-manager.git
```

2. Go to the project folder

```
cd task-manager
```

3. Install required libraries

```
pip install -r requirements.txt
```

4. Run the application

```
python app.py
```

5. Open your browser and go to

```
http://127.0.0.1:5000
```
